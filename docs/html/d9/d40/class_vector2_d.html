Detailed Description

Defines the base mathematical element used, the Vector2D element. Contains structs of two self-contained vector types, the xyVector and the rtVector. Defines the associated creation of the objects, and the logical operator overload associated with vector mathematics. The tigonometric mathematical vector relationships are based on radians.
